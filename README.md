#  Heart Disease Risk Prediction using Machine Learning

##  Project Overview
This project predicts the risk of heart disease using Machine Learning algorithms based on patient medical data.

The application takes important medical parameters as input and provides real-time heart disease risk prediction through an interactive Streamlit web application.

The main objective of this project is to support early diagnosis and preventive healthcare decisions using data-driven insights.
